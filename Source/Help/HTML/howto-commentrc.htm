How do I ... Include comments in a resource file

Comments can be included in an exported resource source file by using the Edit | RC Comments menu option which displays the Comments Editor dialogue box.

Enter multi-line comments as required. Do not enter the special /* and and */ comment delimiters as these will be added automatically. Be sure not to use the delimiters within the comment text as this is an error that will not be detected by the Comments Editor.

Note that RC comments are recorded with preferences so use this facility to record any standard comments.
