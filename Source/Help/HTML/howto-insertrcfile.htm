How do I ... Insert code into an existing resource source file

Sometimes you may prefer to insert or update a version information statement in an existing resource source file rather than create a resource file containing only the version information statement.

Version Information Editor offers no direct way to do this, but does provide some help to make the process easy. Here's what to do:

  1. Copy the current document to the clipboard using the Edit | Copy menu option. The documents is converted into a version information statement and placed on the clipboard in plain text format.
  2. Open the resource source file in a text editor. Find the location to insert the version information and delete any existing statement. *
  3. Paste the version information statement the Editor placed on the clipboard into the resource file.
  4. Save the resource file.

* Only one version information statement is permitted in a resource file so it is important that any existing statement is deleted.
