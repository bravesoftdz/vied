String Information Editor

This dialogue box appears when you edit any STRING INFO item for which editing is currently permitted.

The current value of the string is displayed in a multi-line edit box at the top of the dialogue box. The string can be edited as desired. Note: Pressing return doesn't enter a new line of text but may close the dialog box. This is because STRING INFO text must be on one logical line.

Fields can be entered in the text as follows:

  1. Place the cursor where the field is to be entered in the text.
  2. Select the required field from the Insert Field drop-down list. This list contains all valid fields for the STRING INFO item being edited.
  3. Click the Insert button.

Fields can also be entered by typing them directly into the text, enclosing the name by angle brackets. To enter < character in the text it must be entered as <<>.

Click OK to accept your changes.

Note: When OK is clicked the entry is validated. If an unknown field is encountered an error message is displayed. Dismissing the error message returns you to the dialogue box with the erroneous field highlighted ready for correction. If automatic validation is switched on and no text has been entered for a required STRING INFO item then a warning is displayed with the option of ignoring the warning or returning to the dialogue box to enter some text.
