Number Editor

This dialogue box appears when the FIXED FILE INFO File Type item is set to VFT_VXD and you edit the File Sub-Type.

The current value of the item is displayed in hexadecimal format and is highlighted. Either enter a new value or edit the existing one. Numbers can be entered in either decimal or hexadecimal form. Hexadecimal numbers must be preceded by a $ character.

Click OK to accept the entry. If the number is not valid an error message will be displayed.
