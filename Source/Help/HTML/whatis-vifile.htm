What is ... A Version Information file

A version information file is the native file format used by the Version Information Editor to record the version information provided by the user.

These files have a .vi extension and are plain text files that can be examined in a text editor. A simple format, akin to the Windows ini file format, is used and the files can easily be edited by hand.

Version information files contain:

  1. The version information needed to produce a VERSIONINFO resource statement.
  2. The name of any identifier used in the VERSIONINFO resource statement.
  3. Any comments to be included in the resource source file.
  4. Any comments used to document the file itself.
  5. Any metadata used to configure the program.
