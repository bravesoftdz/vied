View RC Statements Dialogue Box

This dialogue box is displayed when the user chooses the File | View RC Statements menu option.

The VERSIONINFO resource statement corresponding to the current document is displayed. The resource statement depends on the current document, the identifier and any resource file comments that have been specified.

The information displayed here is exactly that which will be used to copy to the clipboard (see the Edit | Copy menu item) or to export as a resource source file (see the File | Export menu item).
