What is ... A Resource File

A resource file is a file containing the definition of various Windows resources to be used by an application, DLL or certain other Windows files. Such resources include:

  • String Tables
  • Bitmaps
  • Icons
  • Cursors

Of course resource files can also include version information. Version information in resource files is contained in a VERSIONINFO statement.

Resource files are usually created as resource source (.rc) files and compiled into binary (.res) files by a resource compiler. These binary files are finally linked into applications by a linker.

This program can create resource source files that contain a single VERSIONINFO statement but no other resources. It can also copy a VERSIONINFO statement to the clipboard for insertion into an existing resource file.

If an external resource compiler is available Version Information Editor can use it to create a binary resource file containing just the version information. More information.
