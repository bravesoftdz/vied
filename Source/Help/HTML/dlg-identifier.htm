Identifier Editor

This dialogue box is displayed when the Edit | RC Identifier menu option is chosen. It allows the identifier used in the VERSIONINFO resource to be edited. When the dialogue box appears any existing identifier is highlighted in the edit box. Enter a new identifier or edit the existing one and then click OK to accept the changes.

If the identifier is invalid an error message will be displayed and the dialogue box will not close.

If you don't want to use an identifier then simply delete any existing one and press OK.
