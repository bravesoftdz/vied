User Setup Dialogue Box

This dialogue box allows the user to specify the state in which the editor starts up. It is accessed using the Options | User Setup menu option.

The intial state of the Options | Automatic Validation and Options | Describe File Flags menu options can be specified. Check the appropriate check box to set the state of the corresponding menu item.

Click OK to accept any changes. Note: The changes take effect the next time the program is started - the current state of the menu options is not affected.
