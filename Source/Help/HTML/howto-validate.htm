How do I ... Validate version information as it is entered

Version information is validated as you enter it if the program's automatic validation option is switched on. This is done by toggling the Options | Automatic Validation menu option so that the menu item is checked.

If you prefer this option to always be switched on use the User Setup dialogue box (accessed from the Options | User Setup menu option) and tick the Automatic validation check box. This has an effect the next time the program is started.
