Pick List Editor

This dialogue box appears when one of the FIXED FILE INFO items:

  • File OS
  • File Type
  • File Sub-Type (when File Type = VFT_DRV or VFT_FONT)

or one of the TRANSLATION INFO items:

  • Language
  • Character Set

are selected and you choose the Edit | Current Item menu option (or you double click one of the items).

The dialogue box displays a drop-down pick list of all the valid symbolic constants (for FIXED FILE INFO items) or descriptions (for TRANSLATION INFO items) for the chosen item. You should select the desired value and click OK to accept your choice.
