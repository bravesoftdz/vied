Edit Menu

Current Item (F2) Causes the currently selected item in the main display to be edited. A dialogue box will appear that allows the item to be edited. The form the dialogue box takes depends on the selected item. See How to Edit an Item for more information.
RC Identifier Displays the Identifier Editor dialogue box where the identifier used in the VERSIONINFO resource statement can be edited. Identifiers are not required. To see how any identifier is defined in a resource statement use the File | View RC statements menu option.
RC Comments Displays the Comments Editor dialog box where the comments written to exported resource source files may be edited.
VI Comments Displays the Comments Editor dialog box where the comments written to saved version information files can be edited.
Compiler Output Folder This option displays a dialog box where the folder used to receive compiled binary resource files can be specified. Compiled files are created when the File | Compile menu option is used. If the program has not been set up to use an external resource compiler this menu option will be disabled.
Analyse Analyses the current document for validity. Any errors are listed in the Analysis Errors dialogue box.
Copy (Ctrl+C) Copies the information in the current document to the clipboard as plain text in resource source file format.
