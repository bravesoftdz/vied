How do I ... Comment my version information document

You can specify comments that will be included in saved version information files by using the Edit | VI Comments menu option which displays the Comments Editor dialogue box.

Enter multi-line comments as required. The ; character is used to mark comment lines. Do not enter this yourself at the start of a line as it is added automatically.

Note that VI comments are recorded with preferences so use this facility to record any standard comments. The program defines a standard set of VI Comments that is uses by default. To remove them delete them in the Comments Editor then save your preferences.
